# statistical-data-analysis-project

End-to-end statistical analysis project using Python:

- Descriptive statistics, data distribution analysis, and outlier detection
- Hypothesis testing: Z-tests, non-parametric tests, and median estimation
- Confidence intervals and bootstrap methods
- Correlation analysis (types of correlation, real-world examples)
- A/B test evaluation, statistical significance, and sample size estimation
- Conversion rate and CTR analysis, loyalty program evaluation
- Technologies: Statistics, Python, Jupyter Notebooks, Pandas, NumPy, SciPy, Matplotlib, Seaborn
